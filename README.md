# рҹҺҜ AI Interview Coach

An AI-powered Interview Preparation System that analyzes resumes, extracts technical skills, generates recruiter-style interview questions, and evaluates candidate answers intelligently.

# рҹҡҖ Project Overview

AI Interview Coach helps users prepare for technical interviews by simulating a real interview environment.

The system:
- Extracts text from uploaded resumes
- Detects technical skills automatically
- Generates interview questions based on detected skills
- Evaluates user answers using AI/NLP techniques
- Provides feedback and scoring

This project is designed to improve interview preparation using Artificial Intelligence and Natural Language Processing.
